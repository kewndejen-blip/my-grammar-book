# ­ЪЊю рІерїЇрІЋрІЮ ріарїѕрЅБрЅЦріЊ рЅЁріћ рѕўрІЮрїѕрЅа рІЋрІЇрЅђрЅх
### (Geez Grammar and Qene Knowledge Base)

рІГрѕЁ рЇЋрѕ«рїђріГрЅх рІерїЇрІЋрІЮ рЅІріЋрЅІріЋ ріарїѕрЅБрЅЦрЇБ рІерЅЁріћ рѕўріЋрїѕрІХрЅйріЋ ріЦріЊ рІерїЇрѕЦ ріЦрѕГрЅБрЅ│рІјрЅйріЋ рЅарІўрѕўріЊрІі ріЦріЊ рѕѕрЅ░рїарЅЃрѕџ рѕЮрЅ╣ рЅарѕєріљ рѕўрѕЇріЕ рѕѕрѕЏрїЦріЊрЅх рІерЅ░рІўрїІрїђ рІхрѕе-рїѕрїй (Web App) ріљрІЇрЇб

## ­Ъџђ рІїрЅЦрѕ│рІГрЅ▒ріЋ рѕѕрѕўріГрЇѕрЅх рІГрѕЁріЋріЋ рІГрїФріЉ (Live Link)
­ЪЉЅ **[https://kewndejen-blip.github.io/my-grammar-book/](https://kewndejen-blip.github.io/my-grammar-book/)**

---

## Рюе рІІріЊ рІІріЊ рїѕрїйрЅ│рІјрЅй (Features)

*   **­ЪћЇ рЇѕрїБріЋ рЇЇрѕѕрїІ (Search):** рЅарѕўрЅХрІјрЅй ріерѕџрЅєрїарѕЕ рѕГрІЋрѕХрЅй рІЇрѕхрїЦ рІерѕџрЇѕрѕЇрїЅрЅхріЋ рЅЃрѕЇ рІѕрІГрѕЮ рѕЁрїЇ рЅарЇЇрїЦріљрЅх рѕЏрїЇріўрЅх рІФрѕхрЅйрѕІрѕЇрЇб
*   **­ЪїЎ рІерѕїрѕірЅх рІЋрІГрЅ│ (Dark Mode):** ріарІГріЋріЋ ріЦріЋрІ│рІГрїјрІ│ рЅђріЋріЊ рѕїрѕірЅх рѕѕрѕўрїарЅђрѕЮ рІерѕџрѕўрЅй рІерЅђрѕѕрѕЮ рѕЮрѕГрїФ (Theme) ріарѕѕрІЇрЇб
*   **­ЪЊ▒ рѕѕрѕхрѕЇріГ рІерЅ░рѕўрЅ╗рЅИ (Responsive):** рЅарѕърЅБрІГрѕЇрЇБ рЅ│рЅЦрѕїрЅх ріЦріЊ рі«рѕЮрЇњрІЇрЅ░рѕГ рѕІрІГ рІФрѕѕрѕЮріЋрѕЮ рЅйрїЇрѕГ рІГрѕ░рѕФрѕЇрЇб
*   **­ЪЊџ рІерЅ░рѕЪрѕІ рѕўрѕерїЃ:**
    *   рІЊрЅбрІГ ріарїѕрЅБрЅЦ (ріеріљ рІЮрѕГрІЮрѕЕ)
    *   ріЋрІЉрѕх ріЦріЊ рІ░рЅѓрЅЁ ріарїѕрЅБрЅЦ
    *   рІерЅЁріћ рѕўріЋрїѕрІХрЅй (рѕ░рѕЮріЊ рІѕрѕГрЅЁрЇБ рѕарѕерІЮрЇБ рІѕрІўрЅ░)
    *   рІерїЇрѕЦ ріЦрѕГрЅБрЅ│рІјрЅй рЅарІеріГрЇЇрѕІрЅИрІЇ

## ­ЪЊѓ рІерЅ░ріФрЅ░рЅ▒ рЇІрІГрѕјрЅй
*   `index.html` - рІІріЊрІЇ рІерЅхрѕЮрѕЁрѕГрЅх рїѕрїй ріЦріЊ рѕўрѕерїЃрІјрЅйрЇб
*   `about.html` - рѕхрѕѕ ріарЇЋрѕірігрѕйріЉ ріЦріЊ рѕхрѕѕ ріарІўрїІрїЂ рѕўрѕерїЃрЇб
*   `privacy.html` - рІерїЇрѕІрІіріљрЅх ріЦріЊ рІерЅЁрїѓ рѕўрЅЦрЅх рѕўрїЇрѕѕрїФрЇб

---

## ­ЪЏа рІерЅ░рѕ░рѕФрІЇ рЅа (Built With)
*   **HTML5** - рѕѕрІГрІўрЅх рѕўрІІрЅЁрѕГ
*   **CSS3** - рѕѕрІ▓рІЏрІГріЋ ріЦріЊ рѕѕріаріњрѕюрѕйріЋ
*   **JavaScript** - рѕѕрЇЇрѕѕрїІ ріЦріЊ рѕѕрІ│рѕГріГ рѕърІх рѕ▓рѕхрЅ░рѕЮ

## ­ЪЉц ріарІўрїІрїЁ (Author)
**рІ▓рІФрЅєріЋ ріерІЇріЋ рІ░рїђріЋ**
> "рІерїЇрІЋрІЮ рЅІріЋрЅІріЋ рѕѕрЅхрІЇрѕЇрІх рѕЏрѕхрЅ░рѕІрѕѕрЇЇ ріЦріЊ рІўрѕўріЊрІі рЅ┤ріГріќрѕјрїѓріЋ рЅарѕўрїарЅђрѕЮ рЅ░рІ░рѕФрѕй рѕЏрІхрѕерїЇ рІЊрѕІрѕЏрЅйріЋ ріљрІЇрЇб"
